# ReVert WordPress Theme Project

Custom WordPress theme for **ReVert** - Agricultural biologicals, stimulants, and nutrients.

## 🎨 Design

Inspired by [Bactivate.com.au](https://bactivate.com.au) - featuring a clean, professional design with a natural green color palette.

## 📦 Theme Location

The WordPress theme is located in: [`/revert-theme`](./revert-theme)

## 🚀 Quick Deploy

### Clone to WordPress:
```bash
cd /var/www/html/wp-content/themes/
git clone https://github.com/DriverJoe/REVERT.git revert-theme
chown -R www-data:www-data revert-theme
chmod -R 755 revert-theme
```

### Or Download ZIP:
1. Click the green **Code** button
2. Download ZIP
3. Upload to WordPress: **Appearance → Themes → Upload**
